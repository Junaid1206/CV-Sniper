# CV-Sniper
A high-precision resume auditing engine powered by Gemini 2.5 Flash, designed to 'snipe' skill gaps and align candidate profiles with elite 2026 market standards. CV Sniper is a high-performance career intelligence dashboard that leverages **Google Gemini 2.5 Flash** to provide tactical resume analysis and market-alignment strategies.

##  Key Features
* **Precision ATS Scoring:** Instantly calculates your resume's match against 2026 market standards.
* **Skill Gap Analysis:** Strategically identifies missing keywords and critical domain expertise.
* **Tactical Roadmap:** Generates a 4-week personalized learning path with curated resources.
* **Live Job Deep-links:** Directly connects you to active openings on LinkedIn and Indeed.
* **AI Career Consultant:** An integrated chatbot for real-time strategic guidance.

## Tech Stack
* **Language:** Python
* **LLM:** Google Gemini 2.5 Flash API
* **Frontend:** Streamlit (Modern UI/UX)
* **Data Viz:** Plotly (Interactive Analytics)
* **PDF Processing:** PyPDF2

##  Local Setup
1. Clone the repo: `git clone https://github.com/Junaid1206/CV-Sniper.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `GEMINI_API_KEY` to `.streamlit/secrets.toml`.
4. Run the app: `streamlit run app.py`
